# Loan Default Risk Analysis

## Project Overview

This project analyzes borrower financial data to identify factors associated with loan default risk. The objective is to explore borrower characteristics, credit history, income levels, and loan attributes to generate actionable insights that can support lending decisions and risk management strategies.

---

## Business Problem

Financial institutions face significant losses when borrowers fail to repay loans. Understanding the factors that contribute to loan default can help lenders improve risk assessment, reduce financial exposure, and develop more effective lending policies.

This analysis investigates borrower behaviour and loan performance to identify patterns associated with default risk.

---

## Dataset Overview

The dataset contains borrower demographic information, financial indicators, loan characteristics, and repayment outcomes.

Key variables include:

* Applicant Income
* Coapplicant Income
* Loan Amount
* Loan Term
* Credit History
* Education
* Employment Status
* Property Area
* Loan Status

## Key Findings

* Borrowers with strong credit histories show significantly lower default rates.
* Higher loan-to-income ratios are associated with increased repayment risk.
* Income levels influence borrowing behaviour and loan amounts.
* Risk segmentation helps identify borrowers with elevated default probability.
* Credit history is one of the strongest indicators of loan performance.

---

## Business Recommendations

* Strengthen credit-history screening during loan approval.
* Monitor borrowers with high loan-to-income ratios more closely.
* Develop risk-based lending policies.
* Implement early-warning systems for high-risk borrowers.
* Utilize predictive analytics to support credit-risk assessment.
* Improve borrower education and financial awareness programs.
